Tutorial
Cara Upload ke GitHub
  1. 1

    Buat Repositori Baru

    Buka github.com, klik New repository, beri nama devops-lab, lalu klik Create repository.

  2. 2

    Inisialisasi Git Lokal

    Di folder project, jalankan git init untuk mulai melacak perubahan file.

  3. 3

    Tambahkan dan Commit File

    Jalankan git add . untuk stage semua file, lalu git commit -m "first commit" untuk menyimpan snapshot.

  4. 4

    Hubungkan ke GitHub

    Salin URL repo dari GitHub, lalu jalankan git remote add origin https://github.com/username/devops-lab.git

  5. 5

    Push ke GitHub

    Upload semua file dengan git push -u origin main. Selesai, project sudah ada di GitHub!
